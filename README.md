# [Project] A Funny Flip Coin Game
## Introduction
*有趣的翻金幣遊戲，需要把所有金幣同時翻正，具有一般關卡與困難關卡*<br>
## Environment
- Windows 10*<br>
- QT 5.15.2*<br>
- G++ 9.4*<br>
## Features
- [x] Basic Mode
- [x] Hard Mode
## Implementation
1. **Basic Mode**
   - 均為4x4關卡，由簡單到困難一共20關，可自由選擇關卡
   - 具備倒數計時功能
2. **Hard Mode**
   - 共五個關卡，挑戰完當前關卡才能到下一關
   - 具備提示功能<br>
